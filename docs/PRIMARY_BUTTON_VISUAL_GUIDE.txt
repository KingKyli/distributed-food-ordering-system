╔═════════════════════════════════════════════════════════════════════════════╗
║                          WHEN TO USE                                        ║
╚═════════════════════════════════════════════════════════════════════════════╝

✅ USE PRIMARY BUTTON FOR:

  • Checkout / Place Order               (Main conversion action)
  • Add to Cart / Add to Basket          (Primary add action)
  • Reorder / Order Again                (Quick re-purchase)
  • Submit / Confirm                     (Form submission)
  • Continue / Next                      (Primary navigation)

❌ DON'T USE PRIMARY BUTTON FOR:

  • Cancel / Back                        (Use secondary button)
  • Edit / Delete                        (Use icon or text button)
  • Multiple actions on same screen      (Only ONE primary per screen)
  • Navigation links                     (Use text button)
  • Low-priority actions                 (Use secondary button)
